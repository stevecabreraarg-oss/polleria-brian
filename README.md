# Polleria Brian

PWA para registrar ventas, totales y stock de una polleria.

## Funciones

- Registro de ventas por fecha, producto, cantidad y precio.
- Precios automaticos con promociones por cantidad.
- Totales por dia, semana y mes.
- Control de stock.
- Exportacion para Excel en CSV.
- Exportacion PDF desde imprimir.
- Instalacion en Android como PWA.
